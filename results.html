Step2: Intercomparison

In this step, we calculate the difference map at first to apply the basic statistics. Then, we split the 31 tiles of our work area and clip the two maps to each of them to calculate the covariance matrix between the two maps.

Step3: Preprocessing data for validation

In this step we used the raster calculator do to define the areas where we have built areas in both maps so can use this map later in our validation. Later, we compute the difference between these two maps to obtain a three scenarios map, showing the area where we have buildings in GHS but not Worldpop, areas where we have buildings in Worldpop but not in GHS, and areas where both maps are populated or not populated.

Step4: Validation

This step is the main step in our work, here compared the data in the given two maps with a reference data by using different base maps. Since it’s required to create a sample of 600 points, the work load was divided into 200 sample for each student. To perform the validation process we had to install AcATaMa plugin in QGIS which refers to Accuracy Assessment of Thematic Maps. We used also python script to calculate the accuracy for both maps.
